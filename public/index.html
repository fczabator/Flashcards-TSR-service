Introduction & Getting started

Microservice is built using Express framework. It integrates with Tesseract OCR library for image proccessing tasks and google APIs for translation purposes. Code can be viewed in Github public repository here

OCR

To use OCR feature simply just send a image in a multipart post request at https://flashcards-ocr.herokuapp.com/upload url.
In response you will receive JSON table that contains single words found on the image.
For example: ["milk", "is" "hot"]

Translation

Translation is performed thanks to google API. Currently it supports only english to polish translations.
To use is send post request with JSON body containing words you want to translate.
For example: ["milk", "is" "hot"] will be processed to this: [{word: "milk", translation: "mleko"}, ...]
